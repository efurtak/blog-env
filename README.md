# Setup

```sh
git clone https://github.com/efurtak/blog-env.git
python3 -m venv blog-env/
cd blog-env/
source bin/activate
pip install -r requirements.txt
cd blogprojectdir/
python manage.py runserver
```

# Go to URL

<http://localhost:8000/blog/posts/>
